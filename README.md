# InterviewGPT V5.0

## 项目简介

InterviewGPT 是基于 Ollama + Qwen3 + Gradio 开发的本地 AI 面试系统。

支持多个技术岗位模拟面试，通过大模型自动完成：

* AI面试提问
* AI回答评分
* AI反馈分析
* AI面试报告生成
* 能力画像分析
* 雷达图可视化
* 历史记录管理
* PDF报告导出
* 多岗位面试支持

---

# V5.0 更新内容

## 多岗位面试系统

从单一 AI 岗位升级为多岗位面试平台。

支持岗位：

* AI应用开发工程师
* Python开发工程师
* 数据分析师

后续可扩展：

* Java开发工程师
* 前端开发工程师
* 测试工程师
* 产品经理

## 动态题库加载

系统根据岗位自动加载对应题库。

题库结构：

```python
questions = {
    "AI应用开发工程师":[...],
    "Python开发工程师":[...],
    "数据分析师":[...]
}
```

无需修改主程序即可扩展新岗位。

---

## PDF报告显示岗位信息

生成PDF时自动记录：

岗位名称

面试时长

面试统计

能力画像

AI分析报告

示例：

岗位：Python开发工程师

平均分：8.6

面试时长：6分钟

---

## 历史记录支持岗位分类

历史记录新增：

```json
{
    "job":"Python开发工程师",
    "skill":"FastAPI",
    "score":8
}
```

便于后续统计：

* 不同岗位成绩
* 岗位能力分析
* 岗位成长曲线
